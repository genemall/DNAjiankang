# DNAjiankang
基因商城 微信公众号
git 使用方式（示例）：

1、先安装git软件
2、新建目录codes/dna
3、cd 到该目录，执行：
git clone https://github.com/genemall/DNAjiankang.git
4、克隆代码后在里面修改个人代码，完成后提交如下：
git add -A
git commit -m '注释'
git push

5、其他命令
git pull （拉取远程代码）
git status （查询git状态）
git log （查看提交日志）
git diff （查看改动前后）
git reset （回退）
